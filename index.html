CRDT Collaborative Text Editor

Three independent replicas (site 1, 2, 3), each backed by its own RGADocument, connected through a simulated network that delivers messages with random delay (and can be fully partitioned). Type in any box, or click "Simulate network partition", type in multiple boxes while disconnected, then heal the partition - all replicas converge to the same text no matter what order edits arrive in.
